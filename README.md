# Classic Mathematics Papers

古典数学論文のデジタルアーカイブ

## 概要

このサイトは、ヒルベルト、ヘッケ、リー、ゲンツェンなどの古典的な数学論文を現代的な形式で提供します。

## 収録内容

### ヒルベルト『数論報告』(Zahlbericht)
- 全5部・27章・114節の完全な目次
- スライド資料（第1-3章）
- 折りたたみ可能な詳細目次

### ヘッケ『代数的数論講義』
- 講義ノートと解説

### リー『変換群の理論』
- 第I-III巻

### ゲンツェン『独立公理系の存在について』

## 技術スタック

- **静的サイトジェネレーター**: Hugo
- **テーマ**: hugo-book
- **数式表示**: MathJax 3.x
- **PDFビューア**: PDF.js
- **ホスティング**: GitHub Pages

## ローカル開発

```bash
# サーバー起動
hugo server -D

# ビルド
hugo
```

## ライセンス

このプロジェクトに含まれる古典的な数学論文は、著作権が切れたパブリックドメインの作品です。
